# Axon 3 分钟视频 Demo 脚本

## 视频信息
- **标题**: Axon — 开源 AI 编程助手，支持任意模型
- **时长**: 2-3 分钟
- **目标**: 展示核心功能，让观众想立即试用
- **风格**: 快节奏、无废话、纯实操

---

## 场景 1：安装启动 (20秒)

**画面**: 终端

```bash
npm install -g axon
axon-web
```

**旁白**: "两行命令，开箱即用。打开浏览器就是一个完整的 IDE。"

**画面**: 浏览器打开 localhost:3456，展示 WelcomeScreen

## 场景 2：Web IDE (30秒)

**画面**: 点击"代码"Tab，展示 Monaco 编辑器 + 文件树

**操作**:
1. 打开一个 .ts 文件
2. 展示语法高亮、文件树导航

**旁白**: "不是终端里聊天，是一个真正的浏览器 IDE。Monaco 编辑器、文件树、终端面板——该有的都有。"

**画面**: 切回聊天 Tab

2. AI 开始分析，展示流式输出
3. AI 调用工具（Glob、Grep、Read），展示工具调用面板

**旁白**: "点一个按钮，AI 就开始工作。它不只是聊天——它能读文件、搜代码、执行命令。37 个内置工具随时待命。"

## 录制注意事项

1. **分辨率**: 1920x1080，暗色主题
2. **字体大小**: 适当调大（14-16px），保证视频缩放后可读
3. **录屏工具**: OBS Studio 或 ScreenToGif
4. **无鼠标抖动**: 操作要流畅自然
5. **每个场景单独录**: 后期剪辑拼接，方便重录某个场景
6. **背景音乐**: 选一段轻快的 BGM，不要太吵
