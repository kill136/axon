# Axon Desktop - 使用说明

## 快速开始

1. 解压 `axon-portable.zip` 到任意目录
2. 双击 `Axon.exe` 启动应用
3. 应用会自动启动 Web 服务器并打开浏览器窗口

## 系统要求

- Windows 10/11 (64位)
- 至少 2GB 可用内存
- 至少 1GB 磁盘空间

## 配置

首次启动需要配置 API Key：
- 在应用中点击设置
- 输入你的 Anthropic API Key
- 配置文件保存在 `%USERPROFILE%\.axon\settings.json`

## 功能特性

- ✅ Web IDE 界面
- ✅ 多代理协作系统
- ✅ MCP 工具集成
- ✅ 代码分析和生成
- ✅ 项目蓝图生成

## 故障排除

### 应用无法启动
- 检查是否有杀毒软件拦截
- 尝试以管理员身份运行

### 端口冲突
- 默认端口 3456，如被占用会自动切换
